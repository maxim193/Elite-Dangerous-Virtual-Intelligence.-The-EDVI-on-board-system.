# EDVI — Elite Dangerous Virtual Intelligence
### On-Board System Companion App for Android

EDVI ist eine vollständig offline-fähige KI-Begleiterin für Elite Dangerous
(PS5 Legacy / Horizons / Odyssey). Sie beinhaltet:

- Offline Wake-Word
- Offline Speech-to-Text (Vosk + Whisper tiny)
- Offline TTS (Android)
- Vision-Pipeline für Kampfanalyse
- Datenbanken: EDSM / EDDB
- NLU/Dialog-Kern
- Persönliche VI-Pilotinnen-Persönlichkeit

Alle Builds laufen automatisch über GitHub Actions — kein PC nötig.
